# 🎹 KeyPiano

> A virtual piano built with **HTML, CSS, and JavaScript** that lets you play musical notes using your keyboard or mouse.

![HTML](https://img.shields.io/badge/HTML5-orange?logo=html5\&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-blue?logo=css3\&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-yellow?logo=javascript\&logoColor=black)


## 📖 About

KeyPiano is a simple and interactive virtual piano that recreates the experience of playing a real piano directly in your browser. Each piano key produces a unique sound and can be played either by clicking with the mouse or by pressing the corresponding keyboard key.

This project is perfect for beginners learning JavaScript event handling, DOM manipulation, and audio playback.

---

## ✨ Features

* 🎹 Play piano notes using your keyboard
* 🖱️ Click piano keys with your mouse
* 🔊 Realistic piano sound effects
* ⚡ Instant key response
* 📱 Responsive design
* 🎨 Clean and modern interface
* 💻 No external libraries required

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/usaidkhot159/KeyPiano.git
```

### Open the project

```bash
cd KeyPiano
```

Simply open **index.html** in your browser.

No installation or dependencies are required.

---

## 🎮 How to Use

1. Open the application in your browser.
2. Press the displayed keyboard keys to play notes.
3. Or click the piano keys using your mouse.
4. Enjoy creating your own melodies!
